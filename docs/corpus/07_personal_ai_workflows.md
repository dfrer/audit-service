# Personal AI Workflows -- Practical Guide
## How Individuals Use AI and Where to Improve

### The Personal AI Stack

A well-organized personal AI setup has 3-5 tools with assigned roles:

| Role | Tool Examples | Purpose |
|------|---------------|---------|
| General Chat | ChatGPT, Claude | Everyday questions, writing, analysis |
| Deep Thinking | Claude, ChatGPT o-series | Complex reasoning, structured analysis |
| Code | Cursor, Claude Code, Copilot | Development, debugging, code review |
| Search/Research | Perplexity, Claude with web search | Factual queries with sourcing |
| Creative | Midjourney, Suno, Claude | Images, audio, visual content |
| Local/Private | Ollama + local model | Private queries, coding, sensitive data |

### Personal Workflow Hygiene

#### System Prompts
Define how you want the AI to behave. One per tool or use case.

- Research: "Be thorough, cite sources, flag uncertainty, no hedging."
- Writing: "Write in plain English. No marketing speak. Short sentences."
- Code: "Explain reasoning. Write tests. Flag edge cases."
- Analysis: "Be structured. Use numbered points. No filler."

#### Templates
Save prompts for recurring tasks. Do not re-type them.

- Weekly review template
- Email response template
- Research question template
- Code review template

#### Conversation Management
- Pin or bookmark important conversations
- Use clear titles for conversations
- Do not lose important threads in the daily noise
- Export or copy outputs worth keeping

### Common Personal Failures

1. **Starting from scratch every time:** No saved prompts, no templates, no custom instructions
2. **Tool amnesia:** Not knowing which tool is best for what, switching randomly
3. **Output amnesia:** Good outputs from last week are lost in the conversation list
4. **Subscription waste:** Paying for Pro when Plus would suffice, or vice versa
5. **Verification gaps:** Trusting AI outputs for facts, numbers, or decisions without checking
6. **No boundaries:** Using AI for everything including things it is clearly bad at

### Assessment Questions

1. Can you name the primary role of each AI tool you pay for?
2. Do you have system prompts or custom instructions set up?
3. Can you recreate your best output from last month?
4. Do you know your total monthly AI spend?
5. Do you verify AI outputs before using them for decisions?
6. If your favorite AI tool shut down today, could you continue your work?
7. How many conversations do you start vs how many do you return to?

### Improvement Path

1. **Week 1:** Document current tools, roles, and spend. Cancel unused subscriptions.
2. **Week 2:** Set up custom instructions / system prompts for each primary tool.
3. **Week 3:** Create 3-5 templates for recurring tasks.
4. **Week 4:** Implement a verification habit for AI outputs in important decisions.
